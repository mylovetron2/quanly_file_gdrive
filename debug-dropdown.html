🔍 Debug Dropdown Menu

Hướng dẫn kiểm tra:

  1. Nhấn vào "Admin" menu ở navbar phía trên
  2. Xem console debug ở phía dưới màn hình
  3. Nếu dropdown hoạt động → Vấn đề là browser cache trên site thật
  4. Nếu dropdown KHÔNG hoạt động → Có lỗi khác (xem console log)
Checklist Debug
⚠️ Nếu dropdown KHÔNG hoạt động trên site thật
Thử các bước sau:
  1. Hard Refresh: Nhấn Ctrl + Shift + R (hoặc Ctrl + F5)
  2. Clear Cache: Xóa cache browser (Ctrl + Shift + Delete)
  3. Kiểm tra file: Đảm bảo file navbar.php đã được upload lên server
  4. Kiểm tra Console: Mở DevTools (F12) → Tab Console → Xem có lỗi gì không
  5. Kiểm tra Network: DevTools → Tab Network → Reload → Xem bootstrap.bundle.min.js có load không
