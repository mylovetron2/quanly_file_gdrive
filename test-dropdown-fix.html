Dropdown menu đã được sửa!

Đã thêm các thuộc tính Bootstrap 5 cần thiết cho dropdown.

Chi tiết sửa lỗi

Vấn đề:

Dropdown menu Admin không hoạt động khi nhấn vào mũi tên xuống.

Nguyên nhân:

Thiếu các thuộc tính Bootstrap 5 bắt buộc:

  • aria-expanded="false" - Thuộc tính accessibility cho dropdown toggle
  • aria-labelledby - Liên kết dropdown menu với toggle button

Đã sửa:

1. Admin Dropdown Toggle:
<!-- CŨ (thiếu aria-expanded) -->
<a class="nav-link dropdown-toggle" href="#" id="adminDropdown" role="button" data-bs-toggle="dropdown">

<!-- MỚI (đầy đủ thuộc tính) -->
<a class="nav-link dropdown-toggle" href="#" id="adminDropdown" role="button" data-bs-toggle="dropdown" aria-expanded="false">
2. Admin Dropdown Menu:
<!-- CŨ (thiếu aria-labelledby) -->
<ul class="dropdown-menu">

<!-- MỚI (có aria-labelledby) -->
<ul class="dropdown-menu" aria-labelledby="adminDropdown">
3. User Dropdown (cũng đã sửa):
<a class="nav-link dropdown-toggle" href="#" id="userDropdown" role="button" data-bs-toggle="dropdown" aria-expanded="false">
<ul class="dropdown-menu dropdown-menu-end" aria-labelledby="userDropdown">

Kết quả:

  • ✅ Admin dropdown hoạt động bình thường
  • ✅ User dropdown hoạt động bình thường
  • ✅ Tuân thủ chuẩn Bootstrap 5
  • ✅ Tuân thủ chuẩn accessibility (ARIA)
  • ✅ Hoạt động trên mọi thiết bị

Test ngay:

Thử nhấn vào các dropdown menu ở navbar phía trên:

  • Admin menu - Nhấn vào "Admin" để xem menu con
  • User menu - Nhấn vào "Admin User" để xem menu con

Lưu ý Bootstrap 5:

Khác biệt với Bootstrap 4:
  • Bootstrap 5 dùng data-bs-* thay vì data-*
  • Bắt buộc phải có aria-expanded cho dropdown toggle
  • Nên có aria-labelledby cho dropdown menu (accessibility)
  • Không cần jQuery - dùng vanilla JavaScript
