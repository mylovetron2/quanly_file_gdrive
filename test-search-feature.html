Hướng dẫn sử dụng tìm kiếm

File search.php đã được tạo thành công!

Bạn có thể sử dụng tính năng tìm kiếm file ngay bây giờ.

Cách sử dụng

1. Tìm kiếm từ Navbar

Sử dụng ô tìm kiếm ở góc trên bên phải của navbar:

[Navbar] → [Ô Tìm kiếm file...] → Nhập từ khóa → Enter
2. Truy cập trực tiếp

Mở URL sau:

https://diavatly.cloud/gdrive-manager/views/dashboard/search.php
3. Tìm kiếm với từ khóa

Thêm tham số ?q= vào URL:

https://diavatly.cloud/gdrive-manager/views/dashboard/search.php?q=nghị bù
4. Ví dụ tìm kiếm
  • Tìm "doc"
  • Tìm "pdf"
  • Tìm "2026"
  • Tìm "biên bản"

Tính năng

  • Tìm kiếm theo:
    • Tên file (file_name)
    • Tên gốc (original_name)
    • Mô tả (description)
  • Hiển thị kết quả:
    • Danh sách file tìm thấy
    • Thông tin chi tiết (kích thước, loại, thư mục, ngày tải)
    • Link đến thư mục chứa file
  • Thao tác trên kết quả:
    • Xem file (PDF, Word, Excel, PPT, ảnh)
    • Tải xuống
    • Sửa mô tả
    • Xóa file
